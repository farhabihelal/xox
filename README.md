# XOX

This is a mini project that uses both Python and Rust.

## Requires

- Python >=3.10
- Rust >=1.86
- `uv`

## Install `uv`

```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## How to Use

```bash
./run.sh
```

## Components

| Name       | Lang   |
| ---------- | ------ |
| UI         | Python |
| Game Logic | Python |
| AI         | Rust   |
